## `print(value)`

Outputs a value to stdout. Works with all types:

```js
print(42)           # 42
print("hello")      # hello
print(true)         # true
print(null)         # null
print([1, 2, 3])    # [1 2 3]
```

`print` accepts multiple arguments and prints each on a new line. It returns the last argument.
